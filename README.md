# Folded-Cacsode-Amplifier
This repository contains the design and simulation of a single-ended Folded Cascode Operational Transconductance Amplifier (OTA) . The circuit was designed and characterized using **Cadence Virtuoso**.  The primary objective of this project was to achieve Unity Gain Bandwidth (UGB) of 10 MHz while maintaining  stability and a high open-loop gain.
